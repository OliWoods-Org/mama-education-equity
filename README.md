> **12 million students** lack adequate tutoring access. Private tutoring costs **$40-100/hour.** The achievement gap isn't about intelligence — it's about access. AI tutoring can close it.

## The Solution

6 COPPA-compliant agents that adapt to each student's learning style, identify gaps, and provide personalized education — for free.

## Agents

**TutorBot** — Adaptive tutoring across subjects, Socratic method
**CurriculumAdapter** — Adjusts difficulty and style to student's pace
**GapDetector** — Identifies knowledge gaps from assessment results
**TeacherAssistant** — Helps teachers create differentiated materials
**ParentDashboard** — Progress reports and engagement suggestions
**ResourceFinder** — Free educational resources, scholarships, programs

## Safety

**COPPA compliant:** Parental consent required for students under 13. No student data sold. Ever.

## Data Sources

State education standards, Khan Academy-style content frameworks, NCES data
